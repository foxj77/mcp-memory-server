A RemoteMCPServer named "{{ .Values.kagent.serverName }}" has been created in
the "{{ .Values.kagent.namespace }}" namespace. Add it to each agent's tool
list based on that agent's required access level:

  # Full read + write  (resolver / advisor agents)
  - type: McpServer
    mcpServer:
      apiGroup: kagent.dev
      kind: RemoteMCPServer
      name: {{ .Values.kagent.serverName }}
      toolNames:
        - create_entities
        - create_relations
        - add_observations
        - search_nodes
        - open_nodes
        - read_graph

  # Observe + read  (analyst agents — can add facts, cannot restructure graph)
  - type: McpServer
    mcpServer:
      apiGroup: kagent.dev
      kind: RemoteMCPServer
      name: {{ .Values.kagent.serverName }}
      toolNames:
        - add_observations
        - search_nodes
        - open_nodes
        - read_graph

  # Read-only  (general-purpose agents — discovery and recall only)
  - type: McpServer
    mcpServer:
      apiGroup: kagent.dev
      kind: RemoteMCPServer
      name: {{ .Values.kagent.serverName }}
      toolNames:
        - search_nodes
        - open_nodes
        - read_graph

To verify the RemoteMCPServer was registered:

  kubectl get remotemcpserver -n {{ .Values.kagent.namespace }} {{ .Values.kagent.serverName }}
